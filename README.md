# Site Futurista com Celular 3D

Projeto criado para testar um modelo 3D `.glb` exportado do Blender usando Three.js.

## Como usar

1. Exporte seu celular no Blender:
   - File
   - Export
   - glTF 2.0
   - Format: glTF Binary (.glb)

2. Salve o arquivo com este nome:

```txt
celular.glb
```

3. Coloque o arquivo dentro da pasta:

```txt
assets/celular.glb
```

4. Abra a pasta no VS Code.

5. Clique com o botão direito no `index.html` e escolha:

```txt
Open with Live Server
```

## Importante

Não abra dando dois cliques no HTML. Use Live Server, porque o navegador pode bloquear o carregamento do arquivo `.glb`.
